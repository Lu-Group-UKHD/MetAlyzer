# Get example log2fc data

This function returns the log2fc dataframe of the Metalyzer_demo
dataset_biocrates MxP Quant 500 XL_2025-04 file, created with the
MetaVizPro package.

## Usage

``` r
toy_diffres()
```

## Value

toy_diffres.rds file path

## Examples

``` r
fpath <- MetAlyzer::toy_diffres()
```
