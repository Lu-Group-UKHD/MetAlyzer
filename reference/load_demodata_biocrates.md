# Get demodata from biocrates

This function returns the Metalyzer_demo dataset_biocrates MxP Quant 500
XL_2025-04.xlsx file path.

## Usage

``` r
load_demodata_biocrates()
```

## Value

Metalyzer_demo dataset_biocrates MxP Quant 500 XL_2025-04 file path

## Examples

``` r
fpath <- MetAlyzer::load_demodata_biocrates()
```
